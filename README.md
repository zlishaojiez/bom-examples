# bom-examples

BOM (Bill of Materials) with Maven Aggregator (Multi-module) Project examples

## flatten-maven

The Maven Aggregator Project version is handled by flatten-maven-plugin

## maven-release

The Maven Aggregator Project version is handled by maven-release-plugin

## Notice

The current repository contains multiple example projects.  
It is recommended to separate your projects into individual repositories, with each project residing in its own dedicated space.
